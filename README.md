# Fuqi_web
云南富齐制冷实业有限公司企业官网静态项目。

## 项目结构

```text
Fuqi_web/
  index.html
  styles.css
  script.js
  assets/
  富齐制冷公司网站项目撰写.md
```

## 页面模块

- 首页
- 关于我们
  - 企业简介
  - 企业文化
  - 企业资质
  - 实用新型专利
- 产品中心
- 合作案例
- 联系我们

## 本地预览

直接双击 `index.html` 即可在浏览器中打开。

## 部署方式

该项目为纯静态网站，可直接上传整个 `Fuqi_web` 文件夹内容到服务器网站根目录。

需要保持以下文件相对路径不变：

- `index.html`
- `styles.css`
- `script.js`
- `assets/`
